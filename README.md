# PythonSimples

Repositório com exercícios e exemplos práticos do curso de Python (Udemy), 
cobrindo desde conceitos introdutórios até uso de Jupyter Notebook.

## 📁 Conteúdo

- `2-dados.py` — exemplos de manipulação de dados
- `3-input.py` — exemplos de entrada de dados via input()
- `exemplo.py` — script de exemplo
- `exemplo2.ipynb` — notebook Jupyter com exercícios
- `home.html` — página HTML de apoio
- `requirements.txt` — dependências do projeto

## 🚀 Como rodar

1. Clone o repositório:
```bash
   git clone https://github.com/gabrielgrifo/PythonSimples.git
```

2. Crie e ative um ambiente virtual:
```bash
   python -m venv .venv
   .venv\Scripts\Activate.ps1
```

3. Instale as dependências:
```bash
   pip install -r requirements.txt
```

4. Para rodar os notebooks, use Jupyter:
```bash
   jupyter notebook
```
